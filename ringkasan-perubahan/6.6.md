# Tahap 6.6 — Dataset Latih ML Pindah ke Folder Host (Transparan & Best-Practice)

**Tanggal Perubahan:** 16 Juni 2026

---

## Ringkasan

User tidak bisa menemukan foto data latih karena tersimpan di **named volume
Docker** (tak terlihat di File Explorer). Dipindahkan ke **folder host** agar
transparan, mudah diaudit, dan mudah di-backup — **tanpa kehilangan data**.

---

## Pertanyaan User yang Dijawab

> "Foto dari user tersimpan di folder mana? Aku ga bisa menemukannya."

Jawaban: foto tersimpan di DUA tempat berbeda:
1. **Foto scan asli** → `storage/app/public/visueco-scans/` (host, terlihat).
2. **Dataset latih ML** → dulu di named volume Docker
   `/var/lib/docker/volumes/.../visueco-ml-data/` (di dalam VM WSL2, **tak
   terlihat di File Explorer** — itu sebabnya tak ditemukan).

Tahap ini memindahkan #2 ke folder host yang terlihat.

---

## Migrasi Data (Tanpa Kehilangan)

Langkah hati-hati agar 61 foto + model tidak hilang:

```
1. mkdir ml-service/model_store               (folder host baru)
2. docker cp visueco-ml:/app/model_store/.     → salin SEMUA isi ke host
   (dataset/1..5, head.keras, meta.json)
3. docker-compose: named volume → bind-mount folder host
4. hapus deklarasi named volume visueco-ml-data
5. docker compose up -d visueco-ml             (recreate dgn mount baru)
```

**Verifikasi pasca-migrasi:**
```
/model/info → version 3, accuracy 0.6, 61 foto utuh, has_trained_head=true ✓
prediksi    → jalan (confidence 0.67) ✓
write test  → container bisa tulis ke folder host (konfirmasi warga tetap jalan) ✓
test suite  → 20 passed ✓
```

## Best Practice yang Dipenuhi

| Praktik | Sebelum | Sesudah |
|---------|---------|---------|
| **Auditable** | ❌ tersembunyi di volume WSL2 | ✅ `ml-service/model_store/dataset/` di Explorer |
| **Backup mudah** | ❌ perlu perintah docker volume | ✅ copy-paste folder biasa |
| **Source of truth jelas** | ⚠️ tak terlihat | ✅ satu folder host |
| Idempotent (hash isi) | ✅ | ✅ (dipertahankan) |
| Tak masuk git | — | ✅ di-gitignore (data runtime) |

---

## Dua Jalur Data Latih (tetap, kini bermuara di folder terlihat)

```
JALUR 1 (manual):
  seed_dataset/<kategori>/  →[tombol admin]→  model_store/dataset/<id>/

JALUR 2 (otomatis):
  warga konfirmasi scan  →[langsung]→  model_store/dataset/<id_benar>/
```

Keduanya kini menulis ke `ml-service/model_store/dataset/` yang **terlihat di
File Explorer**. Foto konfirmasi warga pun langsung muncul di sana.

---

## Catatan untuk User

- **Lihat data latih**: buka `D:\laragon\www\visualEco\ml-service\model_store\dataset\`
- **Backup**: salin folder `model_store` ke tempat aman
- **Reset model**: hapus isi `dataset/`, `head.keras`, `meta.json`, restart ML
- Foto scan mentah tetap terpisah di `storage/app/public/visueco-scans/`
