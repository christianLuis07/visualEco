## Arsitektur Route Web

```
GET  /login       ── guest ── AuthController@showLogin
POST /login       ── guest ── AuthController@login
GET  /register    ── guest ── AuthController@showRegister
POST /register    ── guest ── AuthController@register
POST /logout      ── auth  ── AuthController@logout
GET  /dashboard   ── auth  ── dashboard view
```

Middleware `guest` mencegah user yang sudah login mengakses form login/register. Middleware `auth` melindungi dashboard dan logout.

## Aspek Keamanan UI/UX

### 1. Error Generik — Mitigasi Username Enumeration

```php
// AuthController@login
->withErrors(['email' => 'Email atau password salah.']);
```

Pesan error **tidak membedakan** apakah yang salah adalah email atau password. Jika sistem mengembalikan "Email tidak terdaftar" vs "Password salah", penyerang bisa memetakan daftar email valid yang terdaftar di platform (OWASP A07:2021 — Identification and Authentication Failures). Pesan generik tunggal menutup celah enumerasi ini.

### 2. Token CSRF — Mitigasi Cross-Site Request Forgery

```blade
<form method="POST" action="{{ route('login') }}">
    @csrf   {{-- Menyematkan token tersembunyi --}}
```

Directive `@csrf` menghasilkan `<input type="hidden" name="_token" value="...">` yang berisi token unik per session. Laravel middleware `VerifyCsrfToken` memvalidasi token ini di setiap POST request. Tanpa token yang cocok, request ditolak dengan status 419 — mencegah form submission palsu dari domain eksternal.

### 3. Session Regeneration — Mitigasi Session Fixation

```php
// Login: regenerate session ID setelah autentikasi berhasil
$request->session()->regenerate();

// Logout: invalidasi session + regenerasi CSRF token
$request->session()->invalidate();
$request->session()->regenerateToken();
```

Session ID baru digenerate setelah login untuk mencegah Session Fixation Attack, di mana penyerang memaksa korban menggunakan session ID yang sudah diketahui penyerang.

### 4. Password Hashing — Bcrypt via Hash::make()

Password disimpan menggunakan `Hash::make()` (bcrypt, cost factor 12 sesuai `.env`). Tidak pernah disimpan dalam bentuk plain text.
