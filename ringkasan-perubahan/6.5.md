## 1. Fix Bug Stratified Split (val_accuracy palsu)

### Gejala
Pelatihan pertama pada 59 foto asli: `train 0.63` tapi `val 0.0`. Tidak normal.

### Akar Masalah
Keras `validation_split=0.2` mengambil **20% data TERAKHIR tanpa mengacak**.
Karena dataset tersusun berurutan per kategori (semua plastik → kertas → … →
organik), validation set jadi **hanya 1 kategori** (organik). Model dilatih pada
kategori 1-4 lalu divalidasi hanya pada kategori 5 → akurasi val jatuh ke 0.

### Perbaikan
Stratified split manual di `trainer.py`:
- Acak indeks **tiap kelas** (seed tetap=42 agar reproducible).
- Ambil 20% dari **setiap** kelas untuk validasi (proporsional & seimbang).
- Beri ke Keras via `validation_data` eksplisit, bukan `validation_split`.
- `class_weight` dihitung dari distribusi data train aktual.

### Hasil
```
Sebelum fix: train 0.63  val 0.00  (palsu)
Sesudah fix: train 0.63  val 0.54  (jujur)
Latih ulang: accuracy 0.60  (version 3)
```

## 2. Tombol "Impor Seed + Latih" di Panel Admin

### Alur
```
[Admin klik "Impor Seed + Latih"]
   → POST /admin/model/seed-train
   → ModelTrainerService::seedAndTrain()
       ├─ AiPredictorService::seedDataset()  → ML POST /seed  (impor foto)
       └─ ModelTrainerService::train()        → ML POST /train (latih)
   → catat model_versions, tandai sampel used_in_training
   → tampilkan: "Berhasil mengimpor N foto dan melatih model. Versi vX — akurasi Y%"
```

### Dua Tombol
- **"Impor Seed + Latih"** (ghost) — impor foto baru dari folder seed lalu latih.
- **"Latih Ulang Model"** (primary) — latih dari dataset yang sudah ada saja.

Keduanya berbagi fungsi `runTraining()` di `admin.js` (DRY, saling menonaktifkan
saat proses berjalan).

### Idempotent
Impor seed aman diklik berkali-kali — foto sama tak digandakan (hash isi file).
Bukti uji: `seed_added: 0, seed_skipped: 59` saat folder sudah terimpor.

## Status Model Saat Ini (dari 59 foto aslimu)

| Metrik | Nilai |
|--------|-------|
| Versi | 3 |
| Akurasi (val) | ~54-60% |
| reliable | true (5 kategori, ≥5 foto each) |
| Total foto | 59-61 (12/12/12/11/12 + konfirmasi) |

> **Jujur:** 54-60% adalah titik awal yang wajar untuk ~12 foto/kategori. Untuk
> akurasi tinggi (>80%) butuh lebih banyak foto, terutama variasi yang sering
> tertukar (kaca bening vs plastik bening).

### Cara Menaikkan
1. Tambah foto ke `ml-service/seed_dataset/<kategori>/`.
2. Buka `/admin`, klik **"Impor Seed + Latih"**.
3. Cek akurasi baru di kartu Status Model AI.
