## Ringkasan Logika Bisnis

### 1. Form Request — Penyaringan Layer Terluar
`ScanTrashRequest` memvalidasi bahwa field `image` wajib ada, bertipe gambar (jpeg/png/jpg), dan maksimal 4 MB. Jika gagal, respons langsung dikembalikan dalam format JSON 422 (bukan redirect HTML) agar konsisten untuk konsumsi frontend/mobile.

### 2. Anti-Fraud Check
Setelah AI menganalisis gambar, controller memeriksa dua kondisi:
- `is_recyclable` harus bernilai `true`
- `confidence_score` harus >= 0.60

Jika salah satu gagal, proses dihentikan dan tidak ada poin yang diberikan. Ini mencegah penyalahgunaan berupa upload gambar non-sampah untuk mengumpulkan poin.

### 3. Transaksi Atomik (DB::transaction)
Tiga operasi berikut dibungkus dalam satu transaksi database:
1. Penyimpanan file gambar ke disk (`storage/app/public/visueco-scans/`)
2. Insert record `waste_scans` (log audit hasil AI)
3. Insert record `point_ledgers` (mutasi poin bertipe credit)
4. Increment `points_balance` pada tabel `users` via method `increment()` yang menggunakan query `UPDATE ... SET points_balance = points_balance + ?` — aman dari race condition tanpa perlu locking manual.

Jika salah satu operasi gagal, seluruh transaksi di-rollback dan tidak ada data parsial yang tersimpan.

### 4. Error Handling Berlapis
- **AiServiceException** (503) — koneksi ke AI timeout atau respons gagal
- **Throwable** (500) — error tak terduga lainnya, dilaporkan ke log via `report()`

## Kontrak Response API

### Endpoint
```
POST /api/v1/scan
Header: Authorization: Bearer {token}
Body: multipart/form-data — field "image"
```

### Sukses — 201 Created
```json
{
    "success": true,
    "message": "Scan berhasil diproses dan poin telah ditambahkan.",
    "data": {
        "scan_id": "9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d",
        "detected_item": "Botol Plastik PET",
        "category_name": "Plastik",
        "confidence_score": 0.92,
        "is_recyclable": true,
        "instructions": [
            "Kosongkan isi botol",
            "Lepaskan label dan tutup botol",
            "Remas botol untuk menghemat ruang",
            "Masukkan ke tempat sampah plastik"
        ],
        "points_awarded": 10,
        "points_balance": 150
    }
}
```

### Gagal Validasi — 422 Unprocessable Entity
```json
{
    "success": false,
    "message": "Validasi gagal.",
    "errors": {
        "image": [
            "The image field is required."
        ]
    }
}
```

### Gagal Anti-Fraud — 422 Unprocessable Entity
```json
{
    "success": false,
    "message": "Objek tidak dikenali sebagai sampah yang dapat didaur ulang.",
    "data": {
        "detected_item": "Kucing",
        "confidence_score": 0.95,
        "is_recyclable": false
    }
}
```

### Gagal Koneksi AI — 503 Service Unavailable
```json
{
    "success": false,
    "message": "Koneksi ke server AI terputus atau timeout."
}
```

### Error Internal — 500 Internal Server Error
```json
{
    "success": false,
    "message": "Terjadi kesalahan internal pada server."
}
```
