## Catatan Teknis Penting

### View cache + OPcache `validate_timestamps=0`

Karena optimasi performa Tahap 6.1 (`opcache.validate_timestamps=0` + view cache),
perubahan blade **tidak langsung tampil**. Setelah mengedit view harus:

```bash
docker exec visueco-app php artisan view:clear
docker exec visueco-app npm run build      # agar Tailwind men-scan kelas baru
docker restart visueco-app                  # reset OPcache compiled view
docker exec visueco-app php artisan view:cache
```

Langkah ini sudah dijalankan; landing page terverifikasi tampil (title "Visueco",
font Fraunces, seluruh section ter-render, kelas Tailwind ter-generate di CSS).

### Aset Tailwind

Kelas arbitrary (`bg-[var(--teal-deep)]`, `rounded-[2rem]`, dll.) memerlukan
`npm run build` ulang agar masuk ke `public/build/assets/*.css`. Sudah dibangun
ulang & diverifikasi (backdrop-blur, grid-cols-12, animate-ping, dll. ter-generate).
