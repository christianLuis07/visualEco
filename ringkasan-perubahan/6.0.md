## Ringkasan Eksekutif

Tiga hal besar diselesaikan sesuai permintaan:

1. **Model AI kini BELAJAR** dari tiap foto yang dikonfirmasi/dikoreksi warga
   (transfer learning: MobileNetV2 beku + head classifier yang dilatih ulang).
2. **Semua layanan pihak ketiga DIHAPUS** (AWS/SES/Postmark/Resend/Slack/S3) —
   100% self-hosted.
3. **Bug kategori salah** (botol kaca → Plastik) kini bisa dikoreksi warga, dan
   model belajar dari koreksi itu.

> Menggantikan dokumen rencana sebelumnya. Pipeline `learn → train → predict`
> sudah diuji end-to-end dan berfungsi (lihat bagian Verifikasi).

## Arsitektur Pembelajaran

```
1. Warga upload foto → /predict → tebakan kategori
2. Dashboard tampilkan: "Apakah ini Plastik?" [Ya, benar] [Bukan, koreksi ▼]
3. Warga konfirmasi/koreksi → POST /api/v1/scan/confirm
        │
        ├─ Laravel ambil gambar tersimpan dari disk
        ├─ Kirim ke ML /learn (gambar + kategori benar)
        └─ Catat ke tabel training_samples
4. Admin klik "Latih Ulang Model" → POST /admin/model/train
        │
        ├─ ML /train: ekstrak embedding semua sampel → latih head Dense
        ├─ Simpan head.keras + meta.json ke volume
        └─ Catat model_versions (versi, akurasi, jumlah sampel)
5. /predict berikutnya pakai head terlatih → makin akurat untuk sampah lokal
```

### Dua Mode Prediksi

| Mode | Kapan | Cara |
|------|-------|------|
| **A. Head terlatih** | Setelah `/train` pertama | Feature extractor 1280-dim → Dense classifier 5 kategori (rata-rata lintas crop) |
| **B. Fallback** | Cold start (belum ada training) | Keyword-map ImageNet + multi-crop TTA (perilaku Tahap 5.2) |

### Kenapa Ringan di CPU?

MobileNetV2 **dibekukan** sebagai feature extractor. Hanya head kecil
(1280→128→5) yang dilatih, dengan embedding di-precompute. Training 12 sampel
selesai dalam hitungan detik — tanpa GPU.

## Cara Pakai (untuk Demo)

1. Warga scan sampah di `/dashboard`.
2. Setelah hasil muncul, warga menekan **"Ya, benar"** atau **"Bukan, koreksi"**
   lalu memilih kategori yang tepat (mis. botol kaca → **Kaca**, bukan Plastik).
3. Konfirmasi tersimpan sebagai data latih.
4. Admin buka `/admin`, lihat **Kartu Status Model AI** (versi, akurasi, jumlah
   data latih, berapa yang belum dilatih), lalu tekan **"Latih Ulang Model"**.
5. Model dilatih ulang; prediksi berikutnya memakai pengetahuan baru.

> **Catatan akurasi:** Saat cold-start (belum dilatih), model memakai MobileNetV2
> generik — botol kaca masih mungkin tertebak Plastik. Setelah warga mengoreksi
> beberapa contoh dan admin melatih ulang, akurasi untuk sampah lokal meningkat.
